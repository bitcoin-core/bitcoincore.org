--- name: settxfee btcversion: 27.0.0 btcgroup: wallet permalink: en/doc/27.0.0/rpc/wallet/settxfee/ --- settxfee amount Set the transaction fee rate in BTC/kvB for this wallet. Overrides the global -paytxfee command line parameter. Can be deactivated by passing 0 as the fee. In that case automatic fee selection will be used by default. Arguments: 1. amount (numeric or string, required) The transaction fee rate in BTC/kvB Result: true|false (boolean) Returns true if successful Examples: > bitcoin-cli settxfee 0.00001 > curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "settxfee", "params": [0.00001]}' -H 'content-type: text/plain;' http://127.0.0.1:8332/
