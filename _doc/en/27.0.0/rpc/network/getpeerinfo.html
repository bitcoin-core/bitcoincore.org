--- name: getpeerinfo btcversion: 27.0.0 btcgroup: network permalink: en/doc/27.0.0/rpc/network/getpeerinfo/ --- getpeerinfo Returns data about each connected network peer as a json array of objects. Result: [ (json array) { (json object) "id" : n, (numeric) Peer index "addr" : "str", (string) (host:port) The IP address and port of the peer "addrbind" : "str", (string, optional) (ip:port) Bind not_publicly_routable) "mapped_as" : n, (numeric, optional) The AS in the BGP route to the peer used for diversifying peer selection (only available if the asmap config flag is set) "services" : (boolean) Whether we relay transactions to this peer "lastsend" : xxx, (numeric) The UNIX epoch time of the last send "lastrecv" : xxx, (numeric) The UNIX epoch time of the last receive "last_transaction" : xxx, (numeric) The UNIX epoch time of the last valid transaction received from this peer "last_block" : xxx, (numeric) The UNIX epoch time of the last block received from this
