## Bitcoin Core 0.15.1

### 背景

開始發布最新版本的 Bitcoin Core，以及需要加入到向後移植的 PR。

### 會議意見

- 此版本的版本號從 0.15.0.2 更改為 0.15.1。
- 討論要加入到向後移植的 PR。向後移植討論可以在 [#11592][] 中找到。
- 討論最終從程式碼庫中移除 OpenSSL。由於 BIP 70 無法移除，因為它用於驗證 https 的憑證。
- 此會議合併的 PR
  - 向後移植 PR [#11592][]
  - 合併 PR [#11593][]
  - 合併 PR [#11560][]
  - 合併 PR [#11100][]
- 審查向後移植並開始 RC 程序
