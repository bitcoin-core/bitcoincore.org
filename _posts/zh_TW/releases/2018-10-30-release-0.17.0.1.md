Bitcoin Core 0.17.0.1 版本現已發布，可從以下位置下載：

  <https://bitcoincore.org/bin/bitcoin-core-0.17.0.1/>

此版本為 0.17.0 提供了一個小錯誤修正。

請使用 GitHub 的問題追蹤器回報錯誤：

  <https://github.com/bitcoin/bitcoin/issues>

如需接收安全性和更新通知，請訂閱：

  <https://bitcoincore.org/zh_TW/list/announcements/join/>

重要變更
===============

解決了 OSX dmg 生成的問題，該問題影響 macOS 10.12 到 10.14，可能導致 Finder 在安裝時崩潰。

對於其他作業系統沒有重大變更，使此版本在其他方面與 [0.17.0](/zh_TW/releases/0.17.0/) 相同。
